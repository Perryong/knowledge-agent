# Raw Source Inbox

Store original source material here before creating summaries, analysis, PRDs, stories, or knowledge notes.

Use:

```text
raw/<category>/YYYY-MM-DD-<slug>/
```

Categories:

- `articles/`
- `pdfs/`
- `images/`
- `audio/`
- `video/`
- `documents/`
- `datasets/`
- `github/`
- `trading/`
- `jobs/`
- `research/`
- `meetings/`
- `ideas/`

Each capture folder should include a `metadata.md` file with source details and links to processed outputs.
